My current project

Grasshopper is an ongoing project that I started during my time at the UNCC Coding Bootcamp.


Grasshopper is a platform that stresses the importance of life long learning. The platfrom enables users to easily sort through a catalog of courses of anything the user wishes to learn. The users also has the ability to create a course based on what they already know, or have learned on the platform.


Grasshopper is a full-stack MERN application. The app is built on top of React and uses MongoDB to store data. We used passport.js and quill.js for certain functionality of the app.
